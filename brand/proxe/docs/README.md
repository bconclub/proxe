# PROXe Dashboard

Standalone PROXe brand dashboard application.

## Setup

1. Copy `.env.local.example` to `.env.local` and fill in your PROXe Supabase credentials
2. Install dependencies: `npm install`
3. Run development server: `npm run dev`
4. Build for production: `npm run build`

## Brand Configuration

- **Brand**: PROXe
- **Theme**: Purple (#8B5CF6)
- **Logo**: PROXe logo
- **Supabase**: PROXe project credentials

## Deployment

Deploy independently from Windchasers dashboard:

```bash
npm run build
npm start
```
