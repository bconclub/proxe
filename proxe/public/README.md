# Public Assets Folder

Place your static assets here, such as:

- **Images**: `proxe-icon.png`, `proxe-logo.svg`, etc.
- **Favicons**: `favicon.ico`
- **Other static files**

## PROXe Icon

To use the PROXe icon on the login page:

1. Upload your PROXe icon file here (e.g., `proxe-icon.png` or `proxe-logo.svg`)
2. Supported formats: PNG, SVG, JPG, WEBP
3. Recommended size: 64x64px or larger (will be scaled as needed)

The icon will be accessible at: `/proxe-icon.png` (or whatever filename you use)
