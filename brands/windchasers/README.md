# Windchasers — brand pack

Aviation career academy. Everything brand-specific lives here (nothing shared).
- `config.ts` — identity, theme, flags (which /modules are on)
- `prompts/` — system · voice · web
- `templates.ts` — WhatsApp/email copy · `brand-facts.ts` — persona facts
- `public/` — logo, icon, sounds · `CHANGELOG.md` — this brand's log
- `.env.local` — secrets (gitignored, local only)
The app = /core + enabled /modules + this pack, selected by BRAND_ID=windchasers.
